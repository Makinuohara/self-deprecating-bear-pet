# 自嘲熊桌宠

一个可爱的桌面宠物应用，基于 Electron 开发。

## 功能特性

### 1. 点击互动
- **单击小熊**：显示马卡龙色系对话框，包含可爱语录
- **右键菜单**：提供多种互动选项

### 2. 窗口拖动
- 左键拖动桌宠可自由移动位置
- 学习模式下同样支持拖动
- 单击和拖动智能区分

### 3. 学习陪伴
- 右键菜单选择"熊认真学习形态"
- 桌宠自动变小，不遮挡屏幕
- 45分钟后自动提醒休息
- 显示学习计时器
- 休息时切换为休息状态 GIF

### 4. 摸鱼预警
- 自动检测哔哩哔哩应用
- 停留超过10分钟触发愤怒形象
- 离开B站后恢复正常状态

### 5. GIF 随机播放
- 正常状态：8张 GIF 随机切换
- 休息状态：3张休息 GIF 随机切换
- 摸鱼预警：4张愤怒 GIF 随机切换
- 支持手动"换一只熊"

### 6. 可爱语录
单击小熊显示的语录：
- "蒸蚌！"
- "今天学习效率超高！"
- "这门课肯定不挂科！"
- "早八不迟到，全勤奖到手~"
- "摸鱼不被抓，学习不落下！"
- "期末绩点UP UP!"
- "平淡的日子也是完美的日子"
- "世界不会崩塌，逃跑只是计划"

### 7. 系统托盘
- 自定义图标显示
- 右键菜单控制显示/隐藏
- 快速切换学习模式

## 安装步骤

### 1. 安装依赖
```bash
cd self-deprecating-bear-pet
npm install
```

### 2. 运行应用
```bash
npm start
```

## 技术栈

- **Electron** - 桌面应用框架
- **HTML/CSS/JavaScript** - 前端界面
- **Node.js** - 后端逻辑
- **Sharp** - 图像处理

## 使用说明

### 基本操作
1. **单击小熊**：显示马卡龙色系对话框，包含可爱语录
2. **左键拖动**：移动桌宠位置
3. **右键点击**：显示功能菜单

### 右键菜单功能
- **熊认真学习形态**：启动学习模式，桌宠变小
- **熊不学了**：退出学习模式，桌宠恢复大小
- **换一只熊**：随机切换当前状态的GIF
- **隐身熊**：隐藏桌宠

### 系统托盘功能
- 右键托盘图标显示菜单
- 显示/隐藏小熊
- 熊认真学习形态/熊不学了
- 退出应用

### 学习模式
- 桌宠自动变小（150x210像素）
- 显示学习计时器
- 45分钟后提醒休息
- 休息时切换为休息状态 GIF

### 摸鱼预警
- 自动检测哔哩哔哩应用
- 停留超过10分钟触发愤怒形象
- 离开B站后恢复正常状态

## 自定义

### 修改语录
在 `src/index.html` 中修改 `randomMessages` 数组：

```javascript
const randomMessages = [
  "你的自定义语录1",
  "你的自定义语录2",
  // ...
];
```

### 修改学习时长
在 `src/main.js` 中修改 `STUDY_DURATION` 常量：

```javascript
const STUDY_DURATION = 45 * 60 * 1000; // 45分钟，单位毫秒
```

### 修改 GIF 轮换间隔
在 `src/index.html` 中修改 `ROTATION_INTERVALS` 配置：

```javascript
const ROTATION_INTERVALS = {
  normal: 5 * 60 * 1000,  // 正常状态：5分钟
  rest: 2 * 60 * 1000,    // 休息状态：2分钟
  warning: 30 * 1000      // 预警状态：30秒
};
```

## 注意事项

1. 首次运行需要安装 Node.js 和 npm
2. 哔哩哔哩检测功能仅支持 Windows 系统
3. GIF 图片已包含在项目中
